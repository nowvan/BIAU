Descritption:
PATROS is a free and clean Responsive HTML5 Template for business / corporate websites. 
It is built using Bootstrap 3.3.2 framework, responsive, HTML5 markup validated, easy to customize, and SEO friendly.

Features:
-One Page Template
-3 Pages included: Homepage, Blog and Single-Post.
-Bootstrap 3.3.2
-HTML5 markup validated
-CSS3 and HTML5 Speed Optimized
-Fully Responsive
-Form Validation
-Ajax Contact Form
-Retina Ready
-Font Awesome Icons
-Cross Browser Compatible (IE 10+, Chrome, FF, Safari, Opera)
